How to fix App Check issues:

  1. Click "Generate Debug Token" to get a debug token
  2. Copy the token displayed above
  3. Go to Firebase Console > Project Settings > App Check
  4. Add this debug token for your Web App
  5. Click "Validate App Check" to verify it's working
